# GitHub Pages Site

このリポジトリは、GitHub Pagesを使用してホストされる静的Webサイトです。

## 🌟 特徴

- **モダンなデザイン**: 美しいグラデーションとガラスモーフィズム効果
- **レスポンシブ**: すべてのデバイスで完璧に動作
- **軽量**: 高速な読み込み速度
- **アクセシブル**: すべてのユーザーに優しい設計

## 📁 ファイル構成

```
github-pages-site/
├── index.html      # メインページ
├── about.html      # Aboutページ
└── README.md       # このファイル
```

## 🚀 GitHub Pagesでの公開方法

1. GitHubリポジトリの設定ページに移動
2. "Pages" セクションを選択
3. Source を "Deploy from a branch" に設定
4. Branch を "main" に設定
5. Folder を "/ (root)" または "/github-pages-site" に設定
6. "Save" をクリック

## 🌐 アクセス

GitHub Pagesが有効になると、以下のようなURLでアクセスできます：
`https://[username].github.io/[repository-name]/`
